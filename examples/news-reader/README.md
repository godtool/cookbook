## Tell me the news

### Chatting with LFM2-Audio-1.5B

- CLI client
- Server running on Modal

User asks a question with voice that is saved to a local file.
Local file is pushed to Modal volume.
Request sent to server to generate a response, save it on a file in Modal, and return path to the remote file to the CLI.
CLI downloads file and plays it.
